Detailed Description

Одна клетка игрового поля сапёра.

Класс хранит игровое состояние клетки: значение, индекс текстуры, открыта ли клетка и установлен ли на ней флаг. Координаты клетки здесь не хранятся: их вычисляет MinefieldStorage по индексу.
