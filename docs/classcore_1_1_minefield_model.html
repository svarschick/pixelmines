Detailed Description

Qt-модель минного поля для отображения в QML.

Каждая строка модели соответствует одной клетке поля. Модель отдаёт роли для QML-делегатов, обрабатывает клики в координатах viewport и хранит параметры камеры, позволяющие перемещаться по доске большого размера.
