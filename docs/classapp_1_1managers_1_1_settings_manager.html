Detailed Description

Менеджер загрузки, сохранения и публикации настроек.

SettingsManager создаёт SettingsInteractor, регистрирует его в QML-контексте, получает путь к JSON-файлу настроек через ResourceManager и синхронизирует значения между JSON и интерактором.
