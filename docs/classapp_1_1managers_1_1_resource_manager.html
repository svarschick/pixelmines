Detailed Description

Центральный менеджер внешних ресурсов приложения.

ResourceManager связывает загрузчик файлов с интеракторами QML. После выбора набора ресурсов он обновляет свойства TextureInteractor и отправляет сигналы изменения через IUpdateInteractor.
